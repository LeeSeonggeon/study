1. 출발지 -> 도착지
2. 거리는 상관 없음
3. 좌표에 적인 값 = 도로 복구 시간
4. 가장 적은 시간이 걸리도록 해야 한다. 

BFS 
 넓이 먼저
 모든 간선의 가중치가 동일할 때만 최단 거리를 보장
 FIFO 큐 -> 먼저 도달한 순서로 처리
 미로 탈출, 최단 거리(동일 가중치) 
 이미 방문한 곳은 dist 나 vist배열을 이용해서 방문을 표시
 dist는 순서대로 값(1,2,3,4,...)을 넣어서 이미 값이 있는 곳은 패스 -> 다른 길이 거기를 방문해도 최저 시간을 구하는 거기 때문에 의미 없음

DFS
 깊이 먼저
 모든 경로의 탐색을 진행한다. -> 최단 경로와 어울리지 않음
 가지치기를 하는 if문을 걸어도 모든 경로를 탐색하는 경우도 있을 수 있음
 사용: 조건 만족 경로에 사용, 
 
다익스트라
 BFS의 변형
 항상 정확한 최단 거리 보장
 탐색 방식:  비용 낮은 곳 우선 탐색
 사용: 최단 시간/거리(가중치가 다른 경우)
